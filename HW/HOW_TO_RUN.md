# Инструкция


Данная инструкция позволяет запустить сервис Botify и провести эксперимент с помощью модуля sim

* Я запускал все на MacBook Pro M1
* Возможно некоторые команды нужно будет запускать от имени администратора через sudo

## Botify

C Botify все как обычно:
   
1. Переходим в директорию botify
   ```
   cd ../botify
   ```
   
2. Запускаем сервис
   ```
   docker-compose up -d --build 
   ```

## Sim

На MacBook Pro M1 не запускалось по инструкции из-за поломанных зависимостей, поэтому идем через докер. 
В контейнере автоматически прогонятеся 1000 эпизодов на 8 процессах

1. Переходим в директорию sim
   ```
   cd ../sim
   ```
2. Собираем контейнер
   ```
   docker build -t sim .
   ``` 
3. Запускаем контейнер (предварительно должен быть запущен botify)
   ```
   docker container run --network="host" -t sim   
   ```

# Finish experiment
После того как sim отработал нужно выгрузить логи в папку (например в /tmp/) с помощью команды:
